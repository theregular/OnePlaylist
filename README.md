# OnePlaylist

OnePlaylist is an web app to help you easily create and manage playlists that include songs from multiple music streaming platforms.

The goal is to reduce platform lock-in and give users a unified way to organize and share their music, regardless of where it lives.

## Features

- Authenticate with supported platforms

- Create cross-platform playlists

- Sync and manage tracks without manually recreating playlists on each service
